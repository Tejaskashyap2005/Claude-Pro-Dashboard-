📚 How to add a Claude Pro account to this dashboard

1
Get a Claude Pro subscription

Go to claude.ai and sign up or upgrade an account to Pro plan ($20/month). Each account gets its own token limit (currently ~45 messages per 5 hours for Sonnet, less for Opus).

2
Note the login credentials

Save the email and password for that Claude Pro account. This is what employees will use to log in when they need tokens.

3
Click "Add account" on this dashboard

Enter a friendly name (e.g. "Marketing Team Claude"), the login email, the department, and set the Claude link to https://claude.ai/login — this is the URL employees will click to open and log in.

4
Managing token status

When someone runs out of tokens, they click the toggle button (▲/▼) on the card to mark it as "Exhausted." Other team members can then see which accounts are still available. When the token limit resets (typically every 5 hours), toggle it back to "Available."

5
Employees pick an available account

Employees visit this dashboard, filter by "Available", and click "Open Claude ↗" on any green account. They log in with that account's credentials and start working. Simple as that.
